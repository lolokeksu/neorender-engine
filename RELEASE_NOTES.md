# NeoRender Engine v1.0.0

First public release by **Lolokeksu** for Realme GT Neo 5 SE (`RMX3700` / `RMX3701`) on Android 13.

Highlights:

- transparent Stock, SkiaGL and SkiaVK HWUI profiles;
- safe per-application renderer sessions;
- paired A/B measurements and recommendations;
- OEM baseline restoration and renderer-profile quarantine;
- APatch-tested diagnostics and runtime integrity verification;
- Russian interactive Termux menu;
- no APK, closed binary, telemetry or remote code download.

Global SkiaVK remains experimental on Realme UI. The default profile is Stock.
