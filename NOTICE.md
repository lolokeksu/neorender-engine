# Notice and attribution

NeoRender Engine is an independent clean implementation by Lolokeksu.

The project does not distribute the abandoned closed `core-render` binary or its legacy `Toast.apk`. Runtime behavior is implemented in readable POSIX shell scripts.

Documentation organization was informed by publicly available project documentation from LSPosed, Advanced Charging Controller, Play Integrity Fork and BusyBox for Android NDK. No source code from those projects is included.

Primary technical references for implementation are Android Open Source Project HWUI sources and the official Magisk module documentation.
