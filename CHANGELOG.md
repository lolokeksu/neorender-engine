# Changelog

## v1.0.0 — first public release

- Published NeoRender Engine as an independent project maintained by Lolokeksu.
- Added strict Realme GT Neo 5 SE (`RMX3700` / `RMX3701`) and Android 13 validation.
- Kept Stock as the safe default and global SkiaVK as an explicitly experimental profile.
- Added Stock, SkiaGL and SkiaVK global profile management.
- Added crash-safe per-application renderer transactions.
- Added paired SkiaGL/SkiaVK A/B measurements and fingerprint-bound recommendations.
- Added OEM baseline capture, uninstall restoration and renderer-profile quarantine.
- Added APatch-safe diagnostics, local reports, history, support bundles and runtime integrity checks.
- Added the Russian `neorender` interactive terminal menu.
- Added English and Russian GitHub documentation, security guidance, issue templates and reproducible build scripts.
- Confirmed that the release contains no legacy `core-render` ELF binary, `Toast.apk`, network downloader or telemetry.
